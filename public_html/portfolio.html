Connect Four

In my CSCE 314 (Programming Languages class) project, I had the exciting opportunity to develop an interactive game using JavaFX Scene Builder.

You can view this project on my GitHub and watch the demo video here!

Unlike typical two-player games, this project offered a unique twist: players could challenge an AI opponent with customizable difficulty levels, choosing between "Easy" and "Hard" modes.

In "Easy" mode, the game's moves were primarily randomized, creating an unpredictable experience. On the other hand, "Hard" mode featured a more sophisticated AI that made intelligent decisions. The AI followed a strategic approach, prioritizing moves in the following order:

- Attempting to make a winning move if available.

- Blocking the opponent from achieving a winning move.

- Placing a pawn strategically to disrupt areas with consecutive pawns.
